Optimisation Project


Took an existing site with performance issues and optimised it to achieve a PageSpeed score of 97, running at a smooth 60 fps. To achieve this I used the task runner Gulp to minify and concatenate the HTML, CSS and JavaScript files, as well as minifying the images without quality loss. For the Pizzeria page I investigated bottlenecks thoroughly in Chrome Developer Tools, then employed the latest optimisation techniques to get all animations to run at 60 fps.
To check out my PageSpeed score, paste the site URL onto the PageSpeed Insights website. Only server side optimisations prevent a perfect 100.

ABOUT ME

I am a British front-end web developer based in Japan. In 2016 I completed Udacity's Front-End Web Developer Nanodegree, an online program designed by top Silicon Valley companies Google, AT&T, GitHub and Hack Reactor. Through this and many other modes of learning I have a strong foundation in the fundamental languages of HTML5, CSS3, JavaScript and PHP. I also have experience of using many common web technologies, frameworks and libraries such as jQuery, Bootstrap, Knockout, third-party APIs, GitHub and Gulp. All my work is fully responsive and is meticulously tested on a range of devices. I understand the importance of speed and implement this through optimization techniques and the use of best practices in architecture and design. I am experienced with developing sites for small businesses using WordPress. I am interested in projects worldwide of any kind, so please drop me an email with your requirements!
